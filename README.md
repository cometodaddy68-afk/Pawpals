# PawPals India — Zero-Investment Dropshipping Store

Complete pet products dropshipping store. Free to deploy, zero investment required.

## How to Deploy (Free)

### Option 1: GitHub Pages (Recommended)

1. Create a GitHub account at https://github.com
2. Create a new repository named `yourusername.github.io`
3. Upload all files from this folder to the repository
4. Go to Settings → Pages → Deploy from main branch → Save
5. Your store is live at `https://yourusername.github.io`

### Option 2: Netlify (Drag & Drop)

1. Go to https://netlify.com
2. Drag this entire folder onto the Netlify dashboard
3. Your store is live immediately

## How to Source Products

1. Find products on **AliExpress**, **Amazon**, or **Meesho**
2. Copy product details and images
3. Add pricing (your selling price = cost + 30-50% margin)
4. When customer orders, place the order on the supplier site with customer's address

## Managing Orders

Orders are stored in your browser's localStorage. To view them:
- Open browser DevTools (F12) → Application → Local Storage
- Key: `pawpals_orders`
- Export by copying the JSON data

For a more permanent solution, sign up for **Formspree** (free) and update the checkout form action.

## Customization

- Edit `js/data.js` to add/remove products
- Edit `css/style.css` to change colors, fonts, etc.
- Update contact info in the footer HTML files
